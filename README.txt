This is a backend solution for a Webshop, developed as part of a "Clean Code" course. It emphasizes clean coding principles with a strong foocus on SOLID-principles and Test Driven Development (TDD). Key features in the solution is Generic Repository pattern, Observer pattern and Stategy pattern. There is an In memory database for simulating database management, Swagger for API documentation and testing and is containerized by Docker.
